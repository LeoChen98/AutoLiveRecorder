# AutoLiveRecorder
本程序是直播录播软件，原理为扒流，通过极小的占用完成录播工作，无需打开直播页面也无需留守。目前只支持B站直播。

#### 当前版本：1.0.2.16 Release。

#### 下载地址：http://update.zhangbudademao.com/112/AutoLiveRecorder.exe

代码中FLV前缀的类均为FLV文件处理，但这个版本没有用到。原因是扒流得到的flv文件最后一个Tag不完整无法自动合并。
